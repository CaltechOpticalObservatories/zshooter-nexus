Classical Programs
==================

.. container:: zs-lead

   ZShooter's classical-observing story is as boring as possible in the best sense: clear planning tools,
   low-overhead mode changes, understandable calibration expectations, and quick feedback that lets an observer make
   decisions without guesswork.

   As the project evolves this page will become a long-term home for observer checklists and instrument master
   procedures as the site grows into working nexus.
